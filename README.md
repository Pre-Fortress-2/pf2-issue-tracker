# Pre-Fortress 2 Issue Tracker
Reporting a bug will **require a GitHub account**. 

Every bug report should have a brief title about the bug and a **detailed description** along with what version of the game you're playing. Please check for duplicates reports before submitting a new bug report.


[Click here to submit a bug report.](https://github.com/Pre-Fortress-2/pf2-issue-tracker/issues) Press **New Issue**.
